## What's Next?

### For Future Course Generation

When you're ready to automate course generation:

1. **Get a Gemini API Key**:
   - Go to: https://makersuite.google.com/app/apikey
   - Create a new API key (free tier available)
   - Copy the key

2. **Add to Supabase**:
   - Go to Supabase Dashboard → Settings → Edge Functions
   - Add environment variable: `GEMINI_API_KEY`
   - Paste your API key

3. **Set Up Daily Generation**:
   - Use the edge function `generate_global_daily_courses`
   - Set up a cron job to run it daily at midnight
   - Or manually invoke it each day
